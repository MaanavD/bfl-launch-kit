## What's New

FLUX.2 is a ground-up architecture change, not an incremental update. The short version:

- **Multi-reference control.** Attach up to 10 images. The model preserves identity, layout, and design language across generations. Consistent characters, products, and styles without fine-tuning.
- **Photorealism and detail.** A meaningful jump in fine detail (fabric textures, architectural elements, skin tone) that closes the gap with real photography.
- **Production-ready text.** Typography, infographics, and UI mockups that render legibly. Multilingual.
- **Exact color matching.** Specify `#FF6B35` and get `#FF6B35`. Hex codes integrate into lighting, materials, and reflections. No approximations.
- **Grounding search.** FLUX.2 [max] can search the web at generation time to visualize real-world subjects, trending products, or current events without manual reference sourcing.
- **Enhanced prompt following.** Better accuracy on complex, structured instructions. 32K token input context.
- **Pose guidance and spatial control.** JSON-based controls for positioning, generative expand/shrink, and coherent scene physics.
- **Open weights.** [klein] 4B ships Apache 2.0. Run it locally on ~13GB VRAM.

## Multi-Reference Control

Previous image models had no memory between API calls. Keeping a character consistent across scenes meant training a LoRA: GPU hours, curated datasets, cold starts, retraining when base weights shift. FLUX.2 replaces that workflow for most use cases: attach reference images, and the model locks in identity on the fly. LoRAs still make sense for deeply specialized aesthetics, but the consistency problem that drove most fine-tuning is now solved at inference time.

### Try it: the comic strip test

Give three models the same character reference and ask for three panels. Same person, different scenes:

**Panel 1:** "A woman with short red hair and round glasses sits at a messy desk, coding on a laptop. Overhead fluorescent lighting, office setting, graphic novel style."

**Panel 2:** "The same woman stands in a rainy street at night, holding an umbrella, neon signs reflecting in puddles. Graphic novel style."

**Panel 3:** "The same woman sits in a sunlit cafe booth, laughing while holding a phone. Warm morning light, graphic novel style."

With FLUX.2, attach the character reference once. Face, glasses, and hair stay locked across all three panels. Without reference image support, the character drifts between panels. Run it and compare.

## Text and Color Precision

**Typography.** FLUX.2 renders multilingual text across multi-line layouts, varied font weights, and dense body copy. Headlines, subheadings, weather boxes, columnar layouts. Not the garbled letterforms you're used to from other models.

**Color.** Hex codes in, exact colors out. `#FF6B35` in your prompt produces `#FF6B35` in the image, integrated into lighting and materials. For brand work, e-commerce, and design systems, this eliminates the post-processing loop.
